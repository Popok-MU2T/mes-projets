Configuration Firebase
À faire une seule fois. Tes données seront synchronisées automatiquement sur tous tes appareils sans jamais ressaisir quoi que ce soit.
1
Va sur console.firebase.google.com → Add project → donne un nom → désactive Google Analytics → Create project.
2
Build → Realtime Database → Create database → choisis europe-west1 → Start in test mode → Enable. Puis dans l'onglet Rules, colle :
{ "rules": { "users": { "$uid": { ".read": "auth != null && auth.uid == $uid", ".write": "auth != null && auth.uid == $uid" } } } }
3
Build → Authentication → Get started → Sign-in method → Email/Password → active le premier toggle → Save.
4
⚙ Project Settings → General → scroll → Your apps → icône </> → enregistre l'app → dans le SDK snippet, copie le contenu du bloc firebaseConfig = { … } et colle-le ci-dessous.
