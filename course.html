Configuration Firebase
💡 Utilise le même projet Firebase que l'app Projets — colle la même config, tu te connecteras avec le même compte. Les données course sont stockées séparément et n'interfèrent pas.
1
Va sur console.firebase.google.com → ton projet existant (ou crée-en un nouveau).
2
Si nouveau projet : active Realtime Database (europe-west1) et Authentication → Email/Password. Règles DB :
{ "rules": { "users": { "$uid": { ".read": "auth != null && auth.uid == $uid", ".write": "auth != null && auth.uid == $uid" } } } }
3
⚙ Project Settings → General → Your apps → </> → copie le bloc firebaseConfig = { … } et colle-le ci-dessous.
